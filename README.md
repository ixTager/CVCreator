# CVCreator

## 📚 О проекте

### Сайт для создания резюме онлайн. Пользователь вводит нужные данные и по кнопке `Save` получает готовый файл, формата .pdf.

## ⚒️ Стек

- ### React
- ### Tailwind
- ### MobX

## 🚀 Запуск сайта

### Чтобы запустить данный сайт, введите команду `npm run dev' в консоль.
